Sobre o Projeto

Esta página integra um modelo de reconhecimento de imagens treinado com o Google Teachable Machine diretamente no navegador, sem necessidade de servidor.

Como funciona

O modelo é carregado via TensorFlow.js. A câmera captura frames em tempo real e o modelo classifica o padrão identificado com sua probabilidade.
